Do you like KaseMania? If yes, add this webpage to your favorites to have 1.000 years of luck!

KaseMania is a project where any user can add a project to this website, and there can be a maximum of 96 projects posted. Every time there's a new project posted but there are no more project "slots" available, the project with the worst recession will be deleted and replaced with the new project. So, the project that will last longer is the one with better recensions.
